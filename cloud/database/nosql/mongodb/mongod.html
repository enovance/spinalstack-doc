Class: cloud::database::nosql::mongodb::mongod

Install a MongoDB server & the replicasets

Parameters:

enable

(optional) Should mongod be running. Defaults to 'true'

replset

(optional) MongoDB replicaset to configure Define the replset to enable on the mongodb server Example:

{ 'ceilometer' => { 'members' => '10.0.0.1:27017' }}

Defaults to {}

mongod_port

(optional) Port for the firewall to enable Based on the mode the mongod process is started with, the port it will listen on might change. Defaults to '27017'

firewall_settings

(optional) Allow to add custom parameters to firewall rules Should be an hash. Defaults to {}
