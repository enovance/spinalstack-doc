(optionnal) Hash of checks and their respective options Defaults to {}. Example :

$checks = {
  'ntp' => {
     'command' => '/etc/sensu/plugins/check-ntp.sh'},
  'http' => {
     'command' => '/etc/sensu/plugins/check-http.sh'},
}
